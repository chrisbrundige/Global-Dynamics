Directions for Use:

Log in to application using company supplied username and password.
Press buttons to perform actions. Add customer, adds a customer
delete , deletes etc. Follow prompts on forms to complete actions

Each page of the applications provides CRUD interfacing for a SQL database specific to one or more tables.
Use the menu to navigate to the appropriate page for the action you wish to perform.

To generate reports press the reports link on the left hand side of the screen and choose an appropriate report.

Additional Features:
